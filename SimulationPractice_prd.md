理虚实一体化课程平台需求文档--虚拟演练部分

# 课程名称
新能源汽车动力系统与维护

# 页面布局
页面分四个一级tab，分别为：理论学习、虚拟演练、实训项目、技能认证
四个一级菜单在顶部横排摆放，可点击切换，默认选中”理论学习“

# 虚仿演练tab
给出五个虚仿演练资源的入口，并配图，配图要求为显示虚拟仿真特色。仅第五个资源支持点击，跳转链接为：
1.结构原理虚仿演练
2.拆装虚仿演练-更换充配电总成
3.高压虚仿演练
4.故障维护虚仿演练
5.交流异步电机拆卸虚仿演练

2. 5个虚仿资源详细定义
资源1：动力电池系统虚拟仿真
| 字段             | 内容                                                         |
| ---------------- | ------------------------------------------------------------ |
| **资源名称**     | 动力电池系统结构认知与故障诊断虚拟仿真                       |
| **副标题**       | 3D虚拟拆装与BMS故障排查                                      |
| **封面配图**     | 动力电池包3D爆炸图，展示电芯、模组、BMS、冷却管路等内部结构  |
| **资源分类标签** | 核心系统 / 3D拆装 / 故障诊断                                 |
| **难度等级**     | ⭐⭐⭐ 中级                                                     |
| **建议学时**     | 4学时                                                        |
| **资源简介**     | 通过三维虚拟现实技术，真实还原动力电池系统内外部结构。学员可进行电芯模组拆装、BMS电路检测、热失控故障模拟等操作，掌握动力电池工作原理、关键参数检测及常见故障诊断方法。 |
| **学习目标**     | ①识别动力电池各组成部件 ②掌握BMS通信协议检测 ③完成绝缘故障诊断流程 |
| **技术亮点**     | WebGL 3D渲染 / 真实物理碰撞 / 多工况模拟                     |
| **跳转链接**     | `#/simulation/battery-system`                                |
| **状态标识**     | 未开始 / 学习中 / 已完成                                     |

4. 交互逻辑
| 交互场景       | 行为定义                                           |
| -------------- | -------------------------------------------------- |
| **Tab切换**    | 点击顶部Tab切换，虚拟演练页保持滚动位置记忆        |
| **卡片悬停**   | 封面图放大1.05倍，显示"预览"和"开始"两个按钮       |
| **点击卡片**   | 跳转至对应虚仿资源全屏页面（新标签页或当前页全屏） |
| **筛选标签**   | 点击分类标签，卡片列表动态过滤，有过渡动画         |
| **进度同步**   | 从虚仿资源返回后，自动刷新该卡片的学习进度         |
| **响应式适配** | 窗口缩放时，网格自动从3列→2列→1列调整              |
